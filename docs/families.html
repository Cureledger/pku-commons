Why we built this

If you are here, you already know the routine: read the label, do the math, guess at the recipe, weigh the portion, log it, for almost everything except pure sugar and fat. The routine is relentless, and many families quietly stop logging once they can eyeball a handful of crackers. When that happens, the clinic loses the picture.

Phe-estimator apps can take the labor out of that grind. But new apps are appearing fast, and there has been no way to tell which ones are actually accurate or to know an app will still be around next year.

What PKU Commons does for you:

  • Checks the numbers. Every tool is scored against a public accuracy test, so the claim "it's accurate" is something you can verify yourself.
  • Keeps tools alive. Apps run on shared, open infrastructure, so a good tool doesn't vanish when one developer moves on.
  • Listens to you. Your feedback and your daily reality steer what gets built next.

How we check an app is reliable

In one sentence: we don't take the app's word for it. We test it.

1 · A shared answer key

We build a list of real food labels with the correct phe worked out from trusted public nutrition databases. Anyone can double-check the answers.

2 · Every tool takes the same test

Each app's estimates are compared to that answer key. We measure how close it gets and how often it lands within a safe range.

3 · Scores are public

Results are posted openly and updated over time, so you can see which tools are improving and trust that a tool has to earn its accuracy.
